RouteTrack Scrollable Sidebar Fix

Problem:
The sidebar contains many new pages, but the navigation area could not scroll.
Lower menu items and the account/sign-out section could become hidden.

What this fix does:
- Makes only the navigation menu scroll
- Keeps the RouteTrack logo visible
- Keeps the user profile and Sign out controls visible
- Works on desktop and mobile
- Adds a visible but slim scrollbar
- Supports touch scrolling on phones

Installation:
1. Extract this ZIP.
2. Copy src/app/globals.css into the existing RouteTrack project.
3. Choose Replace the file in the destination.
4. Run:
   npm run build
5. Push to the same repository:
   git add .
   git commit -m "Fix scrollable sidebar navigation"
   git push origin main
6. Wait for Vercel to become Ready.
7. Refresh using Ctrl + Shift + R.

No npm install is required.
No Firestore Rules update is required.
